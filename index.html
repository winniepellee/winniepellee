製卡暨退卡明細自動化表單

2024

學生證製卡與退卡流程涉及多欄位資料整合，傳統手動輸入易導致資料缺漏與格式不一致。透過 AI 協作開發 Excel VBA 工具，操作人員貼入基礎資料後，系統自動帶入函數、驗證欄位完整性、格式化明細輸出，並標記異常項目，有效縮短作業時間並提升資料品質。

核心邏輯 — VBA 自動化片段
' 自動帶入函數並驗證欄位完整性
Sub AutoFillAndValidate()
    Dim ws As Worksheet
    Dim lastRow As Long
    ws = ThisWorkbook.Sheets("製卡明細")
    lastRow = ws.Cells(ws.Rows.Count, "A").End(xlUp).Row

    Dim i As Long
    For i = 2 To lastRow
        ' 檢查必填欄位是否空白，空白則標記紅色警示
        If ws.Cells(i, 2).Value = "" Then
            ws.Cells(i, 2).Interior.Color = RGB(255, 200, 200)
        Else
            ws.Cells(i, 2).Interior.ColorIndex = xlNone
        End If
        ' 自動帶入日期格式
        ws.Cells(i, 5).NumberFormat = "yyyy/mm/dd"
    Next i
    MsgBox "驗證完成，異常欄位已標記"
End Sub

各校學生證掛失資料標準化工具

2024

彙整多所學校提交的掛失資料時，各校格式不一造成後端系統無法統一處理。透過 AI 協作設計 Excel VBA 工具，自動識別並轉換各式輸入格式為統一標準，包含欄位對齊、資料型態轉換、重複值偵測與缺漏提示。

核心邏輯 — VBA 重複值偵測與格式統一
' 偵測重複身份證字號並統一格式
Sub NormalizeAndDedup()
    Dim ws As Worksheet
    Dim dict As Object
    dict = CreateObject("Scripting.Dictionary")
    ws = ThisWorkbook.Sheets("掛失資料")
    Dim lastRow As Long
    lastRow = ws.Cells(ws.Rows.Count, "A").End(xlUp).Row

    Dim i As Long
    For i = 2 To lastRow
        Dim idNo As String
        idNo = UCase(Trim(ws.Cells(i, 1).Value))  ' 統一轉大寫並去空白
        ws.Cells(i, 1).Value = idNo

        If dict.Exists(idNo) Then
            ws.Cells(i, 1).Interior.Color = RGB(255, 220, 100) ' 重複標黃
        Else
            dict.Add idNo, i
        End If
    Next i
    MsgBox "格式標準化完成，重複項目已標記"
End Sub
